# Day 1 Notes

## Terminal commands
- pwd: shows the current working directory
- cd: changes into another directory
- mkdir: creates a new directory
- ls: lists the contents of the current directory
- touch: creates a new empty file

## Git commands
- git init: initializes a folder to be tracked by Git
- git status: shows the current state of the repo - staged, modified, and untracked files
- git commit: saves the staged changes as a permanent snapshot with a message
